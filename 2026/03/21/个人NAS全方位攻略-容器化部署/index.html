  3. е®№еҷЁеҢ–йғЁзҪІпјҲеҪ“еүҚпјү
  4. еёёз”ЁиҪҜд»¶

DockerеҹәзЎҖзҹҘиҜҶиҜ·еҸӮиҖғиҸңйёҹж•ҷзЁӢпјҢиҝҷйҮҢдёҚеҶҚиөҳиҝ°гҖӮжң¬з« еҸӘи®Ёи®әеңЁNASеңәжҷҜдёӢдҪҝз”ЁDockerж—¶йңҖиҰҒдәҶи§Јзҡ„е®һи·өеҶ…е®№гҖӮ

1. дҪҝз”ЁDocker Composeз®ЎзҗҶе®№еҷЁ

еҰӮжһңдҪ жӣҫз»ҸеңЁе‘Ҫд»ӨиЎҢйҮҢз”Ёdocker runеҗҜеҠЁиҝҮе®№еҷЁпјҢдҪ еә”иҜҘдҪ“йӘҢиҝҮйӮЈз§Қз—ӣиӢҰвҖ”вҖ”еҸӮж•°еҸҲй•ҝеҸҲеӨҡпјҢжҜҸж¬ЎйғҪиҰҒзҝ»еҺҶеҸІи®°еҪ•жүҚиғҪжӢјеҮәе®Ңж•ҙзҡ„еҗҜеҠЁе‘Ҫд»ӨгҖӮжӣҙйә»зғҰзҡ„жҳҜпјҢеҪ“дҪ йңҖиҰҒеңЁдёҖеҸ°ж–°жңәеҷЁдёҠйҮҚе»әиҝҷдәӣе®№еҷЁж—¶пјҢеҰӮжһңжІЎжңүжҠҠе‘Ҫд»Өи®°еҪ•дёӢжқҘпјҢеҮ д№ҺдёҚеҸҜиғҪиҝҳеҺҹеҮәе®Ңе…ЁдёҖиҮҙзҡ„й…ҚзҪ®гҖӮ

Docker Composeи§ЈеҶізҡ„е°ұжҳҜиҝҷдёӘй—®йўҳгҖӮе®ғз”ЁдёҖдёӘYAMLж–Үд»¶еЈ°жҳҺејҸең°е®ҡд№үжүҖжңүе®№еҷЁзҡ„й…ҚзҪ®вҖ”вҖ”й•ңеғҸгҖҒз«ҜеҸЈгҖҒжҢӮиҪҪеҚ·гҖҒзҺҜеўғеҸҳйҮҸгҖҒдҫқиө–е…ізі»зӯүпјҢз„¶еҗҺйҖҡиҝҮdocker compose up -dдёҖжқЎе‘Ҫд»ӨеҗҜеҠЁжүҖжңүжңҚеҠЎгҖӮзӣёжҜ”зәҜе‘Ҫд»ӨиЎҢж–№ејҸпјҢе®ғжңүеҮ дёӘжҳҺжҳҫзҡ„дјҳеҠҝпјҡ

  • й…ҚзҪ®еҸҜиҜ»еҸҜз»ҙжҠӨпјҡYAMLж–Үд»¶з»“жһ„жё…жҷ°пјҢдёҖзңје°ұиғҪзңӢеҮәжҜҸдёӘе®№еҷЁзҡ„й…ҚзҪ®
  • еҸҜзүҲжң¬жҺ§еҲ¶пјҡжҠҠcomposeж–Үд»¶ж”ҫиҝӣGitд»“еә“пјҢй…ҚзҪ®зҡ„д»»дҪ•еҸҳжӣҙйғҪжңүи®°еҪ•пјҢйҡҸж—¶еҸҜд»Ҙеӣһж»ҡ
  • дёҖй”®еҗҜеҒңпјҡdocker compose up -dеҗҜеҠЁпјҢdocker compose downеҒңжӯўе№¶жё…йҷӨпјҢдёҚеҶҚйңҖиҰҒи®°дҪҸжҜҸжқЎdocker runе‘Ҫд»Ө
  • дҫқиө–з®ЎзҗҶпјҡеҸҜд»Ҙе®ҡд№үе®№еҷЁд№Ӣй—ҙзҡ„еҗҜеҠЁйЎәеәҸе’Ңдҫқиө–е…ізі»пјҢзЎ®дҝқжңҚеҠЎжҢүжӯЈзЎ®зҡ„йЎәеәҸеҗҜеҠЁ

еңЁNASдёҠйғЁзҪІжңҚеҠЎж—¶пјҢе»әи®®жүҖжңүе®№еҷЁйғҪйҖҡиҝҮDocker ComposeжқҘз®ЎзҗҶгҖӮдёәжҜҸдёӘжңҚеҠЎеҲӣе»әдёҖдёӘзӢ¬з«Ӣзҡ„зӣ®еҪ•пјҢеӯҳж”ҫdocker-compose.ymlе’Ңзӣёе…ій…ҚзҪ®ж–Үд»¶пјҢиҝҷж ·еҚідҪҝNASйҮҚиЈ…дәҶзі»з»ҹпјҢеҸӘиҰҒжҠҠcomposeж–Үд»¶е’ҢжҢӮиҪҪзҡ„ж•°жҚ®зӣ®еҪ•еӨҮд»ҪеҘҪпјҢе°ұиғҪеҝ«йҖҹжҒўеӨҚжүҖжңүжңҚеҠЎгҖӮ

2. ж•°жҚ®жҢҒд№…еҢ–

Dockerе®№еҷЁжң¬иә«жҳҜдёҙж—¶зҡ„вҖ”вҖ”е®№еҷЁиў«еҲ йҷӨеҗҺпјҢе®№еҷЁеҶ…йғЁзҡ„жүҖжңүж•°жҚ®йғҪдјҡдёўеӨұгҖӮеӣ жӯӨпјҢжүҖжңүйңҖиҰҒжҢҒд№…дҝқеӯҳзҡ„ж•°жҚ®йғҪеҝ…йЎ»йҖҡиҝҮжҢӮиҪҪеҚ·пјҲVolumeпјүжҲ–з»‘е®ҡжҢӮиҪҪпјҲBind Mountпјүжҳ е°„еҲ°е®ҝдё»жңәдёҠгҖӮ

дёӨз§Қж–№ејҸзҡ„еҢәеҲ«пјҡ

  • з»‘е®ҡжҢӮиҪҪпјҲBind MountпјүпјҡзӣҙжҺҘе°Ҷе®ҝдё»жңәзҡ„жҹҗдёӘзӣ®еҪ•жҳ е°„еҲ°е®№еҷЁеҶ…гҖӮжҜ”еҰӮжҠҠNASдёҠзҡ„/data/nextcloudжҢӮиҪҪеҲ°е®№еҷЁеҶ…зҡ„/var/www/htmlгҖӮдјҳзӮ№жҳҜзӣҙи§ӮпјҢдҪ еҸҜд»ҘзӣҙжҺҘеңЁNASзҡ„ж–Үд»¶зі»з»ҹдёӯзңӢеҲ°е’Ңз®ЎзҗҶиҝҷдәӣж–Үд»¶
  • DockerеҚ·пјҲVolumeпјүпјҡз”ұDockerз®ЎзҗҶеӯҳеӮЁпјҢж•°жҚ®еӯҳж”ҫеңЁ/var/lib/docker/volumes/дёӢгҖӮдјҳзӮ№жҳҜDockerеё®дҪ з®ЎзҗҶеӯҳеӮЁдҪҚзҪ®пјҢз§»жӨҚж—¶жӣҙж–№дҫҝпјҢдҪҶж–Үд»¶еңЁNASзҡ„ж–Үд»¶зі»з»ҹдёӯдёҚеӨӘе®№жҳ“зӣҙжҺҘжүҫеҲ°

еңЁNASеңәжҷҜдёӢпјҢжҺЁиҚҗдҪҝз”Ёз»‘е®ҡжҢӮиҪҪпјҢеҺҹеӣ еҫҲз®ҖеҚ•пјҡдҪ иғҪзӣҙжҺҘзңӢеҲ°ж•°жҚ®еңЁе“ӘйҮҢпјҢеӨҮд»Ҫе’ҢиҝҒз§»йғҪеҫҲж–№дҫҝгҖӮ

йңҖиҰҒжіЁж„Ҹзҡ„дёҖдёӘеқ‘жҳҜж–Үд»¶жқғйҷҗй—®йўҳгҖӮе®№еҷЁеҶ…иҝҗиЎҢзҡ„з”ЁжҲ·е’Ңе®ҝдё»жңәдёҠзҡ„з”ЁжҲ·еҸҜиғҪдёҚжҳҜеҗҢдёҖдёӘUIDпјҢеҜјиҮҙе®№еҷЁеҶ…еҲӣе»әзҡ„ж–Үд»¶еңЁе®ҝдё»жңәдёҠжІЎжңүиҜ»еҶҷжқғйҷҗгҖӮи§ЈеҶіж–№жі•жҳҜеңЁcomposeж–Үд»¶дёӯйҖҡиҝҮuserеҸӮж•°жҢҮе®ҡиҝҗиЎҢз”ЁжҲ·пјҢжҲ–иҖ…еңЁе®ҝдё»жңәдёҠи°ғж•ҙжҢӮиҪҪзӣ®еҪ•зҡ„жқғйҷҗгҖӮ

3. зҪ‘з»ңй…ҚзҪ®

Dockerе®№еҷЁй»ҳи®ӨдҪҝз”ЁbridgeзҪ‘з»ңпјҢжҜҸдёӘе®№еҷЁжңүиҮӘе·ұзӢ¬з«Ӣзҡ„IPең°еқҖпјҢе®№еҷЁд№Ӣй—ҙйҖҡиҝҮе®№еҷЁеҗҚдә’зӣёи®ҝй—®гҖӮдҪҶеңЁNASеңәжҷҜдёӢпјҢзҪ‘з»ңй…ҚзҪ®жңүеҮ дёӘе®№жҳ“иё©зҡ„еқ‘пјҡ

дёҚиҰҒйҡҸж„ҸдҪҝз”ЁhostзҪ‘з»ңжЁЎејҸ

еҫҲеӨҡж•ҷзЁӢдёәдәҶи®©й…ҚзҪ®з®ҖеҚ•пјҢдјҡжҺЁиҚҗдҪҝз”Ёnetwork_mode: hostпјҢи®©е®№еҷЁзӣҙжҺҘдҪҝз”Ёе®ҝдё»жңәзҡ„зҪ‘з»ңж ҲгҖӮиҝҷж ·зЎ®е®һзңҒеҺ»дәҶз«ҜеҸЈжҳ е°„зҡ„йә»зғҰпјҢдҪҶд»Јд»·жҳҜе®№еҷЁеӨұеҺ»дәҶзҪ‘з»ңйҡ”зҰ»вҖ”вҖ”е®№еҷЁеҸҜд»ҘзӣҙжҺҘи®ҝй—®е®ҝдё»жңәзҡ„жүҖжңүзҪ‘з»ңз«ҜеҸЈпјҢе…¶д»–е®№еҷЁе’ҢжңҚеҠЎзҡ„з«ҜеҸЈд№ҹеҸҜиғҪиў«еҚ з”ЁеҜјиҮҙеҶІзӘҒгҖӮеңЁеҗҢдёҖдёӘNASдёҠиҝҗиЎҢеӨҡдёӘе®№еҷЁж—¶пјҢhostзҪ‘з»ңжЁЎејҸеҫҲе®№жҳ“еј•еҸ‘з«ҜеҸЈеҶІзӘҒй—®йўҳгҖӮ

жӯЈзЎ®еҒҡжі•жҳҜдҪҝз”Ёй»ҳи®Өзҡ„bridgeзҪ‘з»ңпјҢйҖҡиҝҮportsеҸӮж•°еҸӘжҡҙйңІйңҖиҰҒеӨ–йғЁи®ҝй—®зҡ„з«ҜеҸЈгҖӮе®№еҷЁй—ҙйҖҡдҝЎзӣҙжҺҘдҪҝз”Ёе®№еҷЁеҗҚеҚіеҸҜпјҢDockerеҶ…е»әзҡ„DNSдјҡиҮӘеҠЁи§ЈжһҗгҖӮ

з«ҜеҸЈдёҚиҰҒеҶІзӘҒ

жҜҸдёӘйңҖиҰҒеӨ–йғЁи®ҝй—®зҡ„е®№еҷЁйғҪиҰҒжҳ е°„дёҖдёӘе®ҝдё»жңәз«ҜеҸЈгҖӮйғЁзҪІж–°е®№еҷЁеүҚпјҢе…ҲзЎ®и®ӨиҰҒжҳ е°„зҡ„з«ҜеҸЈжІЎжңүиў«е…¶д»–жңҚеҠЎеҚ з”ЁпјҢйҒҝе…ҚеҗҜеҠЁеӨұиҙҘгҖӮе»әи®®дёәеёёз”Ёзҡ„жңҚеҠЎзәҰе®ҡеӣәе®ҡзҡ„з«ҜеҸЈиҢғеӣҙпјҢйҒҝе…ҚеҗҺз»ӯеҶІзӘҒгҖӮ

4. е®үе…ЁдёҺжқғйҷҗ

дёҚиҰҒдҪҝз”ЁвҖ“privileged

--privilegedеҸӮж•°дјҡиөӢдәҲе®№еҷЁеҮ д№ҺзӯүеҗҢдәҺе®ҝдё»жңәзҡ„е®Ңж•ҙжқғйҷҗвҖ”вҖ”еҢ…жӢ¬и®ҝй—®жүҖжңүи®ҫеӨҮгҖҒдҝ®ж”№зі»з»ҹй…ҚзҪ®зӯүгҖӮиҝҷеңЁеҫҲеӨҡж•ҷзЁӢдёӯдҪңдёәвҖқдёҮиғҪи§ЈеҶіж–№жЎҲвҖқеҮәзҺ°пјҢдҪҶе®ғе®Ңе…Ёжү“з ҙдәҶе®№еҷЁзҡ„йҡ”зҰ»жңәеҲ¶гҖӮеҰӮжһңдёҖдёӘе®№еҷЁиў«е…ҘдҫөпјҢж”»еҮ»иҖ…зӣёеҪ“дәҺзӣҙжҺҘжӢҝеҲ°дәҶдҪ NASзҡ„rootжқғйҷҗгҖӮ

еҰӮжһңжҹҗдёӘе®№еҷЁзЎ®е®һйңҖиҰҒи®ҝй—®зү№е®ҡи®ҫеӨҮпјҲеҰӮUSBи®ҫеӨҮпјүпјҢеә”иҜҘдҪҝз”Ё--deviceеҸӮж•°зІҫзЎ®жҢҮе®ҡпјҢиҖҢдёҚжҳҜдёҖдёҠжқҘе°ұз»ҷе…ЁйғЁжқғйҷҗгҖӮ

д»Ҙйқһrootз”ЁжҲ·иҝҗиЎҢ

й»ҳи®Өжғ…еҶөдёӢпјҢе®№еҷЁеҶ…зҡ„иҝӣзЁӢд»Ҙrootиә«д»ҪиҝҗиЎҢгҖӮиҷҪз„¶е®№еҷЁжңүдёҖе®ҡзҡ„йҡ”зҰ»пјҢдҪҶиҝҷд»Қз„¶жҳҜдёҚеҝ…иҰҒзҡ„е®үе…ЁйЈҺйҷ©гҖӮеҫҲеӨҡе®ҳж–№й•ңеғҸйғҪжҸҗдҫӣдәҶйқһrootиҝҗиЎҢзҡ„ж–№ејҸпјҢеңЁcomposeж–Үд»¶дёӯйҖҡиҝҮuserеҸӮж•°жҢҮе®ҡеҚіеҸҜгҖӮ

жңҖе°ҸеҢ–з«ҜеҸЈжҡҙйңІ

еҸӘжҳ е°„жңҚеҠЎе®һйҷ…йңҖиҰҒзҡ„з«ҜеҸЈпјҢдёҚиҰҒдёәдәҶеӣҫж–№дҫҝжҠҠжүҖжңүз«ҜеҸЈйғҪжҡҙйңІеҮәжқҘгҖӮеҶ…йғЁжңҚеҠЎпјҲеҰӮж•°жҚ®еә“пјүдёҚйңҖиҰҒеҜ№еӨ–жҡҙйңІпјҢеҸӘе…Ғи®ёеҗҢдёҖзҪ‘з»ңеҶ…зҡ„е…¶д»–е®№еҷЁи®ҝй—®еҚіеҸҜгҖӮ

5. ж—Ҙеҝ—дёҺиө„жәҗз®ЎзҗҶ

ж—Ҙеҝ—иҶЁиғҖ

Dockerе®№еҷЁзҡ„ж—Ҙеҝ—й»ҳи®ӨжІЎжңүеӨ§е°ҸйҷҗеҲ¶пјҢй•ҝжңҹиҝҗиЎҢзҡ„е®№еҷЁж—Ҙеҝ—дјҡжҢҒз»ӯеўһй•ҝпјҢжңҖз»ҲеҸҜиғҪж’‘ж»ЎдҪ зҡ„зі»з»ҹзӣҳгҖӮиҝҷжҳҜNASдёҠйқһеёёеёёи§Ғзҡ„вҖқзЈҒзӣҳз©әй—ҙзҘһз§ҳж¶ҲеӨұвҖқзҡ„еҺҹеӣ гҖӮ

е»әи®®еңЁcomposeж–Үд»¶дёӯдёәжҜҸдёӘе®№еҷЁй…ҚзҪ®ж—Ҙеҝ—й©ұеҠЁйҷҗеҲ¶пјҡ

иҝҷж ·жҜҸдёӘе®№еҷЁзҡ„ж—Ҙеҝ—жңҖеӨҡдҝқз•ҷ3дёӘж–Үд»¶пјҢжҜҸдёӘж–Үд»¶жңҖеӨ§10MBпјҢдёҚдјҡж— йҷҗеўһй•ҝгҖӮ

иө„жәҗйҷҗеҲ¶

еҰӮжһңдҪ зҡ„NASеҗҢж—¶иҝҗиЎҢеӨҡдёӘе®№еҷЁпјҢжҹҗдёӘе®№еҷЁеҰӮжһңеҮәзҺ°ејӮеёёпјҲжҜ”еҰӮеҶ…еӯҳжі„жјҸпјүпјҢеҸҜиғҪдјҡеҚ з”ЁеӨ§йҮҸзі»з»ҹиө„жәҗпјҢеҜјиҮҙе…¶д»–е®№еҷЁз”ҡиҮіNASжң¬иә«иҝҗиЎҢзј“ж…ўгҖӮе»әи®®еңЁcomposeж–Үд»¶дёӯдёәе…ій”®е®№еҷЁи®ҫзҪ®еҶ…еӯҳе’ҢCPUйҷҗеҲ¶пјҡ

иҝҷж ·еҚідҪҝе®№еҷЁеӨұжҺ§пјҢд№ҹдёҚдјҡеҪұе“ҚNASзҡ„ж•ҙдҪ“иҝҗиЎҢгҖӮ

6. е®№еҷЁжӣҙж–°дёҺз»ҙжҠӨ

е®№еҷЁеҢ–йғЁзҪІзҡ„дёҖдёӘеҘҪеӨ„жҳҜжӣҙж–°ж–№дҫҝпјҢдҪҶд№ҹйңҖиҰҒжіЁж„ҸжӯЈзЎ®зҡ„ж–№ејҸпјҡ

жӣҙж–°й•ңеғҸ

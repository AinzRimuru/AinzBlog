## 连接协议选择
强制使用 UDP，1% 丢包在 TCP 下会被放大为严重的延迟，而 UDP 下则基本无影响。
```
proto udp
```

## 增加缓冲区
根据实际延迟与最大带宽计算出合理的缓冲区大小，单位为字节。
如延迟为100ms，最大带宽为100Mbps，那么缓冲区大小应该为100ms * 100Mbps * 1000 / 8 = 1250000Bytes。对应的配置为：
```
sndbuf 1250000
rcvbuf 1250000
push "sndbuf 1250000"
push "rcvbuf 1250000"
```
上述值还可以适当调大。
